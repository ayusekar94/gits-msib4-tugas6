Name: ``` Ananda Ayu Sekar Wiranti ```

Kampus: ``` Politeknik Negeri Malang ```

## Tools

- laravel
- Boostrap
- Jquery
- BoostrapIcon

## Cara Menjalankan

1. Clone Repo
``` 
git clone https://github.com/ayusekar94/gits-msib4-tugas6
```

2. Copy env dari env.example
3. Generate key
``` 
php artisan key:generate
```

4. Mingrate tabel
``` 
php artisan migrate:fresh --seed
```
